kse.library.types package¶

kse.library.types.int_types module¶

User defined integer types

Any of those types should follow eval(object.__repr__()) == object

Where object is an instance of any of these classes

Examples

eval(IntHexU16(16512).__repr__()) == IntHexU16(16512) IntHexU16(16512) == 16512

Integer factory representation class

This is a subclass of int that represents integers in hex format format by default In other respects it should behave exactly the same as an integer
