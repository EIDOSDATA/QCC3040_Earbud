GraphÂ¶

Graphs are container objects for Endpoint, Stream and Capability.

Suported operations are:

  • kse.kalsim.graph.graph.Graph.load() to load a graph configuration

  • kse.kalsim.graph.graph.Graph.create() to create a graph based on the configuration provided

  • kse.kalsim.graph.graph.Graph.config() to configure a graph

  • kse.kalsim.graph.graph.Graph.connect() to connect all elements in a graph

  • kse.kalsim.graph.graph.Graph.start() to start streaming

  • kse.kalsim.graph.graph.Graph.check_active() to check if streams has finished

  • kse.kalsim.graph.graph.Graph.stop() to stop streaming

  • kse.kalsim.graph.graph.Graph.disconnect() to disconnect all elements in a graph

  • kse.kalsim.graph.graph.Graph.destroy() to destroy a graph

  • kse.kalsim.graph.graph.Graph.play() to play a graph, which essentially is execute all methods above in sequence

In the shell it is available as graph
