class kse.kalsim.graph.graph.Graph(stream, endpoint, capability, kcapability, kymera, config)[source]¶

Bases: object

Kalimba graph handler

This class handles all elements in a class (streams, endpoints, operators and koperators). It has the capability to create, configure, connect, start streaming, stop streaming, disconnect and destroy.

Parameters

  • stream (kats.kalsim.stream.stream_factory.StreamFactory) – Stream factory

  • endpoint (kats.kymera.endpoint.endpoint_factory.EndpointFactory) – Endpoint factory

  • capability (kats.kymera.capability.capability_factory.CapabilityFactory) – Capability factry
