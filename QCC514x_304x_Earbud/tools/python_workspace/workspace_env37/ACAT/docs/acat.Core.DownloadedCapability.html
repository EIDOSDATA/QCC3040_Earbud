DownloadedCapability¶

This module can be imported from ACAT.Core.DownloadedCapability.

Download Capabilities.

This module enables ACAT to recognise downloadable capabilities.

class ACAT.Core.DownloadedCapability.MappedTable(chipdata, debuginfo)[source]¶

Maps global addresses to downloadable capability’s local ones.

This class defines a table of mapped addresses used to convert global addresses of debuginfo and chipdata to local addresses of downloaded capabilities. Full protocol is described in CS-340329-SP.
