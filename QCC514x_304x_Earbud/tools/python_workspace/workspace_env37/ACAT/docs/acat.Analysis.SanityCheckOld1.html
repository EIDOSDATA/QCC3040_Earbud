Prints out all of the memory-mapped registers.

Reg names are not precise (i.e. they are guesses), so this should only really be used in Interactive mode.

analyse_firmware_id(id_mismatch_allowed=None, verbose=True)[source]¶

Check the firmware id for mismatch.

Compare the firmware ID reported in the chipdata with the one in the debug information (checks we have got the correct build output) id_mismatch_allowed is a boolean. If supplied, it overrides any command-line setting.

get_panic_time()[source]¶

Get the panic time.

If we panicked at some point in the past, returns the value of TIMER_TIME at which we did so. If we haven’t ever panicked, returns None.

run_all()[source]¶

Performs analysis and spew the output to the formatter.

It analyses the firmware ID to make sure we supplied the correct build output, if the chip has panicked and it also analyses the Symbol Look-Up Table.
