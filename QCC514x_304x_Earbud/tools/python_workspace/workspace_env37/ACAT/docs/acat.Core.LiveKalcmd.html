get_all_proc_regs()[source]¶

Returns a dictionary containing all processor registers.

Dictionary maps name to value. NB on a running chip there is the obvious risk of tearing. Consider stopping the chip if this is an issue.

get_proc_reg(name)[source]¶

Returns the value of a Processor Register.

Return the value of the processor register specified in name. name is a string containing the name of the register in upper or lower case, with or without the prefix REGFILE_:

e.g. REGFILE_PC, rMAC, R10.
