exceptions¶

This module can be imported from ACAT.Core.exceptions.

Exceptions.

exception ACAT.Core.exceptions.AmbiguousSymbolError[source]¶

Multiple symbol matches were found and couldn’t decide which to return.

When raising the exception, we can choose to pass a simple string argument(as normal for exceptions), or pass a tuple of (string, args_list).

exception ACAT.Core.exceptions.AnalysisError[source]¶

Executing the analysis faces a problem.

An error occurred; we can’t complete this analysis. When raising the exception, we can choose to pass a simple string argument(as normal for exceptions), or pass a tuple of (string, argument).

Signals if the chip type is not yet set internally. This could happen if it is not clear form the coredump or live connection the version of the chip.
