Scheduler Analysis.

Module used to analyse Kymera scheduler.

analyse_tasks()[source]¶

Displays the tasks.

Outputs analysis of all currently-active tasks, as well as any pending messages.

analyse_timers()[source]¶

Analyse event timers.

Outputs analysis of strict and casual event timers, plus takes a guess at what the time was when we panicked.

Get the time of the next timer trigger.

Reads the value of timer registers and returns the time at which the next strict timer will fire, or None if no timer is currently set.

Raise a background interrupt.

Parameters

run_all()[source]¶

Performs analysis and spew the output to the formatter.

It analyses the active tasks and any pending messages, the timers for the casual and strict events, the time a panic occurred and the wake up timer.
