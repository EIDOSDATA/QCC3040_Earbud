class ACAT.Display.HtmlFormatter.HtmlFormatter(file_path)[source]¶

Implements the Formatter interface to provide html output.

Creates table of content for each section. Each section is displayed in different html section with a title. The sections are separated with lines.

class ACAT.Display.HtmlFormatter.Section(title, parent=None, level=1, proc='P0')[source]¶

Class to encapsulate a section.

In html a section is represented as header followed by a series of divs.

Create a new section. When called publicly, the parent and level parameters must not be given and the result will be a top level section. The parameters are filled in when creating subsections which users of this class should do by calling the subsection method on an existing section.

Parameters

  • title (str) – Plain text title.

  • parent – Pointer to the parent object.

  • level (int) – Nesting depth of this object.

  • proc – processor being documented.

subsection(title)[source]¶

Create a subsection of this section.

This creates a new section as a subsection of this one. Once that has been done, the add_report method of this section must not be called. It’s still safe to call add_alert and add_error.
