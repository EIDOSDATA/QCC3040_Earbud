class ACAT.Analysis.DmProfiler.DmProfiler(**kwargs)[source]¶

Encapsulates an analysis for DM memory: find the owner of individual blocks, in heap and pools. Then for each owner report the sum of owned blocks, and a list of allocated blocks for (a list of) owner(s).

Analyse DM memory used by tasks and display it.

Parameters

  • every (bool) – If True, analyse only operator tasks, if False, analyse all tasks.

  • verbose (bool) – if True, show all heap and pool blocks used by a task/bgint if False, don’t show any heap and pool block information

profile_dm_memory(every=False)[source]¶
Analyse DM memory used by tasks and return a list

with profile information. Each list entry is a list itself with N entries:

  1. Task ID,

  2. Task description,

  3. Sum size of allocations (in octets),

  4. Sum size of all heap allocations (in octets),

  5. Sum size of all pool allocations (in octets),

  6. List of the allocated blocks for this entry’s task ID, each of which is a list by itself, e.g.:

[[address1, length1, name1],
 [address2, length2, name2],
 ...
]
Parameters

every (bool) – If True, analyse only operator tasks, if False, analyse all tasks.
