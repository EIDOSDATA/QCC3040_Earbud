block_is_free(block)[source]¶

Checks whether a block is free or not.

Takes a block variable (such as might be obtained via get_block_from_addr) and works out whether or not it’s free. Also takes an address in DM and performs the same function on the block containing that address.

get_dmprofiling_pool_blocks(task_list, tf_list, f_list)[source]¶
Construct a list of pool owner sizes. Each size is the sum

of all pools with a given owner id (for every ‘task_list’ entry).

Parameters

task_list – an ordered list of unique owner ids

It outputs the list of pool owner sizes, the size of all pools with an owner not in ‘task_list’, and a list of descriptions of pool blocks owned by ‘task_list’ entries.

get_pool(pool)[source]¶

Gets a specific memory pool, split into blocks.

Note

‘pool’ is the unique pool ID for a pool; if you want a pool in DM2 (e.g. L_Mem2Pool2) you’ll need to know what that is (e.g. pool=14).

Parameters

pool –

Returns

A nonstandard Variable containing individual blocks of the pool.
