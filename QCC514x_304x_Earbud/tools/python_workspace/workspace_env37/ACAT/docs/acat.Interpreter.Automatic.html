Automatic¶

This module can be imported from ACAT.Interpreter.Automatic.

Automatic Mode Interpreter.

This module implements the automatic mode which runs all the analyses one after the other.

class ACAT.Interpreter.Automatic.Automatic(p0=None, p1=None, analyses=None)[source]¶

Encapsulates an automatic analysis session.

Parameters

  • p0 – A Processor instance.

  • p1 – A Processor instance.

  • analyses (list) – A list of requested analyses. If this option is not provided all the default analyses will be performed.

run()[source]¶

Runs all the available analyses for each processor.
