This module can be imported from ACAT.Analysis.AANC.

Adaptive ANC ACAT Analysis Module.

This module provides an ACAT analysis class and python object representation for AANC operators.

Examples

Display information about the currently running AANC operator(s).

>>> aanc.run_all()

Get a list of currently running AANC operators.

>>> aanc.find_operators()

Find an adaptive ANC operator in the graph.

This can be used if the AANC object was initialized but the graph has changed. It follows the same approach as the analysis aanc.find_operator, so looks for operators with a particular name.
