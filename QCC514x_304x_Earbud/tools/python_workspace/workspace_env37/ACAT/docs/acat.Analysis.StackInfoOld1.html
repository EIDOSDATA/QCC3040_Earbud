get_stack_frames()[source]¶

Get stack frames.

Returns

a list of tuples, each containing a stack frame. The

oldest frame is at the start, and frames are listed with lowest- address first - that is, the same way DM normally appears. Might raise an AnalysisError exception.

Return type

list
