TypeDependencies¶

This module can be imported from ACAT.Core.TypeDependencies.

Module used for testing if the elf is supported.

class ACAT.Core.TypeDependencies.StructureTree(var, indent=0)[source]¶

Structure Tree.

This class takes Variable object and returns its structure representation as a tree, name of variable being a root and members as branches. Every branch of a tree is another tree, thus many layers can exist to fully represent the structure>

Parameters

  • var – Variable structure.

  • indent – Indentation of every layer of the tree for string representation purposes.

compare(tree, equals=True)[source]¶

Compares another Structure Tree instance with this one.

Two tree structures are equal if names and sizes of all branches of a tree are equal.

Parameters

  • equals (bool) –

Returns

contains_member(member_name)[source]¶

Checks if a member is part of the type structure.

This method takes member name and checks whether this member is part of the type structure (represented as a tree). The member name can be a full name containing the whole structure name. e.g. ‘struct ENDPOINT.ep_to_kick’ or just the member name e.g ‘ep_to_kick’.

Parameters

member_name (str) – The member name we will search in the variable.

Returns

True if exists, False otherwise.

Return type

bool
