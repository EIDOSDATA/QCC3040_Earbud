class ACAT.Core.CoreTypes.ChipdataVariable(debuginfo_var, chipdata)[source]¶

A chipdata variable.

Variables are more complicated than a standard data symbol, since they have a size and can be structures.

‘value’ can be a single integer or (more likely) a list/tuple.

Parameters

  • - Debuginfo variable (debuginfo_var) –

  • - chipdata (chipdata) –

property address¶

Returns start address of the variable in the chip memory.

property deref¶

Returns the de-referenced pointer to the variable.

If the ChipdataVariable is a pointer it returns the dereferenced pointer to variable. Otherwise it raises VariableNotPointerError error.
