class ACAT.Interpreter.Interpreter.Interpreter(p0=None, p1=None, analyses=None, analyses_exceptions=None)[source]¶

Base class for all interpreters.

Parameters

  • p0 – A Processor instance

  • p1 – A Processor instance

  • analyses (list) – A list of requested analyses. If this option is not provided all the default analyses will be performed.

  • analyses_exceptions (list) – A list/tuple of analyses which should not be included. This option is more useful if the child class is relying on default analyses.
